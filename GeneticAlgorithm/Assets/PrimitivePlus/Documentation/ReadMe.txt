///////////
// USAGE //
///////////
//
// - In your Hierarchy select Create > Primitive Plus > [Shape]
// - Your new shape will be placed on the scene view's pivot
// - UNITY_5: Right clicking a GameObject will add it as a child GameObject at Vector.zero
// - New shapes use a basic Mesh Collider by default
// - Primitive Plus Material allows you to share or use new materials on each shape
// - - > No need to create a new material, just open up the attached material and change the colour / texture
// - > New Material - makes a new Material default-diffuse or standard(unity5) on the MeshRenderer
// - > Share Material - shares the same material from the first selected game object
//
